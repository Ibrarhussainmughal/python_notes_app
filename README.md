# Python Notes App

A simple, efficient Command Line Interface (CLI) application for managing personal notes, flashcards, and Q&A snippets.

## Features

- **Categorized Storage**: Organize notes by topics (e.g., "Python", "Generators", "Interview").
- **CRUD Operations**:
    - **View**: List all notes by category.
    - **Add**: Create new questions and answers.
    - **Edit**: Update existing answers.
    - **Delete**: Remove obsolete notes.
- **Search**: Quickly find information using keywords.
- **Persistent Storage**: All data is saved locally in `categorized_notes.json`.

## Use Cases

- **Study Aid**: Create Q&A flashcards for exams or learning new subjects.
- **Developer Cheatsheet**: Store syntax, commands, or code snippets you frequently forget.
- **Interview Prep**: Keep a list of common interview questions and your refined answers.
- **Personal Wiki**: Track random facts, definitions, or reminders.

## How to Run

1.  Ensure you have Python installed.
2.  Run the application:
    ```bash
    python notes_app.py
    ```
3.  Follow the on-screen menu to manage your notes.
